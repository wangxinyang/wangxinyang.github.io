kibana設定

ElasticSearch Kibana
Publish Date:   2020-07-23

Kibana設定に関して

  1. インストール

    Elasticsearchと同じバッジョンをダウンロードする。

    自分は6.8.0を使用していますので、kibana-6.8.0-linux-x86_64.tar.gzをダウンロードしました。

    ルートユーザーを利用してtarファイルをアップロードする。

    tar -xvf kibana-6.8.0-linux-x86_64.tar.gz

  2. kibana.ymlを修正する

    // 自分のelasticsearchのホストを設定する
server.host: "192.168.10.100"
elasticsearch.hosts: ["http://192.168.10.100:9200"]

  1. 立ち上げ

    cd /kibana/bin
./kibana

Kibanaのインデックスの基本操作

  • インデックスを新規
PUT /ems
  • 全てのインデックスを獲得
GET /_cat/indices?v
  • インデックスを削除
DELETE /ems
DELETE /*  #全部を削除する
  • typeを設定して新規
PUT /ems
{
  "mappings": {
    "emp": {
      "properties": {
        "id": {"type": "keyword"},
        "name": {"type": "text"},
        "age": {"type": "integer"},
        "bir": {"type": "date"}
      }
    }
  }
}
  • dangdang id name price author des pubdate
PUT /dangdang
{
  "mappings": {
    "book": {
      "properties": {
        "id": {
          "type": "keyword"
        },
        "name": {
          "type": "keyword"
        },
        "price": {
          "type": "double"
        },
        "author": {
          "type": "keyword"
        },
        "des": {
          "type": "text"
        },
        "pubdate": {
          "type": "date"
        }
      }
    }
  }
}
  • query index
GET /dangdang
GET /dangdang/_mapping
  • ドキュメントを新規
PUT /ems/emp/1
{
  "id": 1,
  "name": "田中",
  "age": 23,
  "date": "2020/12/11"
}
  • ドキュメントを検索
GET /ems/emp/1
  • ドキュメントを更新
POST /ems/emp/R3yYenMBt6TDWkHn1NM6/_update
{
  "doc": {
    "name": "原",
    "age": 44,
    "desc": "新規のカラム"
  }
}
  • batch更新(_bulk) 新規(index)　修正(update)　削除(delete)
POST /ems/emp/_bulk
{"index": {}}
  {"name": "大島", "age": 34, "desc": "いい人です"}
{"delete": {"_id": 3}}
{"update": {"_id": 2}}
  {"doc": {"name": "test"} }

Kibanaのインデックスの高級検索

  • 全部のドキュメントを検索してソートする（QueryString）
GET /ems/emp/_search?q=*&sort=age

＃ ageを降順する
GET /ems/emp/_search?q=*&sort=age:desc

# pagination from size
# _source 検索したいカラム
GET /ems/emp/_search?q=*&sort=age&size=2&from=0&_source=name
  • 全部のドキュメントを検索してソートする（QueryDSL）
# 全部を検索する
GET /ems/emp/_search
{
  "query": {
    "match_all": {}
  }
}
  • ソーティング
GET /ems/emp/_search
{
  "query": {
    "match_all": {}
  },
  "sort": [
    {
      "age": {
        "order": "desc"
      }
    },
    {
      "bir": {
        "order": "desc"
      }
    }
  ]
}
  • pagination _source
GET /ems/emp/_search
{
  "query": {
    "match_all": {}
  },
  "from": 0,
  "size": 2, 
  "sort": [
    {
      "age": {
        "order": "desc"
      }
    }
  ],
  "_source": ["name", "age"]
}
  • term検索
GET /ems/emp/_search
{
  "query": {
    "term": {
      "name": {
        "value": "test"
      }
    }
  }
}
